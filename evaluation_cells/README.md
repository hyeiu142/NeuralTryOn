# Evaluation Cells

Reusable cell sources for the SD + LoRA evaluation notebook.

Run through:

```text
notebooks/03_evaluation/model_3_sd_lora_evaluation.ipynb
```

The runner loads these files in order:

```text
cell12_config.py
cell13_load_models.py
cell14_inference_fn.py
cell15_paired_test_metrics.py
cell16_export_paired_three_columns.py
```

Holdout evaluation cells are optional.
