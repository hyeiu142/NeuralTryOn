# Results

Store small, report-ready artifacts in this directory:

```text
eda/             Data quality and preprocessing figures
metrics/         CSV/JSON summaries comparing all models
convergence/     Training and validation loss charts
comparisons/     Qualitative try-on comparisons
error_analysis/  Representative failure cases
```

Large full-resolution result galleries should be published as Kaggle Datasets
and linked from `docs/checkpoints.md` instead of committed to Git.

See `notebook_image_manifest.md` for the source notebook and cell of extracted
figures.

See `archive_image_manifest.md` for Model 3 report artifacts recovered from the
local image archive.
